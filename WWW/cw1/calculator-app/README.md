# Calculator App

This is a simple calculator application that allows users to perform basic mathematical operations.

## Features

- Two input fields for entering numbers.
- A selection field for choosing the mathematical operator (addition, subtraction, multiplication, division).
- A button to calculate the result and display it.

## Usage

1. Open the `index.html` file in a web browser.
2. Enter numbers in the input fields.
3. Select the desired operator from the dropdown menu.
4. Click the "Calculate" button to see the result.

## Technologies Used

- HTML
- CSS
- JavaScript
