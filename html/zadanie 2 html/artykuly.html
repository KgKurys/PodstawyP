Jak napisać funkcję w JavaScript

1 października 2023

Funkcja w JavaScript, która przyjmuje imię jako parametr i zwraca powitanie. Jest to prosty przykład, który pokazuje, jak używać funkcji do generowania dynamicznych wyników:

function greet(name) {
    return `Witaj, ${name}!`;
}

Jak efektywnie używać funkcji w JavaScript

10 października 2023

Funkcja z parametrem domyślnym pozwala na ustawienie wartości domyślnej, jeśli użytkownik nie poda argumentu. Dzięki temu kod jest bardziej elastyczny i odporny na błędy:

// Funkcja z parametrem domyślnym
function greet(name = "Gość") {
    return `Witaj, ${name}!`;
}

// Wywołanie funkcji
console.log(greet()); // Witaj, Gość!
console.log(greet("Grzegorz")); // Witaj, Grzegorz!

Funkcja strzałkowa to nowoczesny sposób definiowania funkcji w JavaScript. Jest bardziej zwięzła i szczególnie przydatna w przypadku krótkich operacji:

// Funkcja strzałkowa
const add = (a, b) => a + b;

// Wywołanie funkcji
console.log(add(5, 3)); // 8
